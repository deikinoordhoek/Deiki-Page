Cool new stuff!
We've made some major modifications that we think make Ubuntu 10.04 LTS the best Ubuntu yet.
Internet
Evolution Email

We've improved Evolution's Microsoft Exchange compatibility, allowing you to easily connect to just about any Exchange server. We've also made Evolution faster, easier to use, and more responsive.

Mobile Internet

Connect to the web wherever you are. Now you can use a mobile broadband modem or supported mobile phone to go online anywhere you have cellphone coverage!

Transmission BitTorrent client

Ubuntu's got a new application this time around; Transmission BitTorrent client. With it, you can download large files much faster than you can through a web browser.

Ubuntu One File Sync

Got multiple computers? Ubuntu One makes it easy to keep all your stuff in sync between them. Just put files into the "Ubuntu One" folder, and they'll be automatically kept updated on all the other computers you connect to your Ubuntu One account. It's that easy!

Firefox
Speed

The fastest version of Firefox yet, 3.6 is three times faster than Firefox 3.0. The web never seemed so fast before.

Personas

Make your browser your own! Choose from any of thousands of Personas to change the look of your browser with one click.

Private Browsing

Sometimes it's nice to go undercover, so turn on Private Browsing and protect your browsing history. It's great if you're doing your online banking on a shared computer or checking email from an Internet cafe.

Web Standards

Firefox has traditionally been the most standards-compliant web browser in existence, and 3.6 takes this to a new level with support for HTML based audio and video, as well as advanced file APIs.

Music
Ubuntu One Music Store

Ubuntu's new music player includes an integrated store, so you can buy and download new tracks with just a few clicks. And thanks to Ubuntu One's file-syncing magic you can store your music online and listen to it from other computers and music players.

Music player support

In addition to a music store, we've added support for even more music players, including the iPod and iPhone! Whatever player you've got, you can be pretty sure it works with Ubuntu!

Sound just works!

Remember how poorly 8.04 performed when playing music or sounds? We're pleased to report that we've solved the sound problems in time for Ubuntu 10.04, so sound now works on almost all computers.

Last.FM

Also new in 10.04, Rhythmbox Music Player can now play Last.FM online radio. Get music from a radio station personalized just for you!

Software Center
A new way to install software
In Ubuntu 8.04, you had to install software through a program called "Add\Remove Programs". But it was seriously overcomplicated - and, we're ashamed to admit, poorly designed.

So now we've created a new way to install software - the Software Center. It's dead simple - just two clicks can install any piece of software. Have a look at the new features.

Simple Interface

We've removed every bit of unnecessary clutter from the Software Center, making it one of the simplest and most useful applications ever made. Now you can install anything in just two clicks!

Install Non-App Packages

The Software Center is capable of installing non-application packages, including fonts, themes, programming libraries, and more!

Subcategories

Applications are now arranged in subcategories as well as categories, making finding software you don't know the name of far easier. Especially useful for games!

Featured Applications

We've selected the best applications from the Center and put them in their own category. Be sure to check it out after you upgrade!

Personal Package Archives (PPAs)

Get apps to your users fast. With personal package archives (PPAs), you can easily upload new or experimental versions of your software for easy installation anywhere. See help.launchpad.net/Packaging/PPA for more information.

Better Descriptions

The new descriptions are longer, better formatted, and contain screenshots, making app selection so much easier.
